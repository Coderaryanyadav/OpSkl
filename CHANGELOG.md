# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Planned
- Multi-language support
- Advanced analytics dashboard
- AI-powered worker-gig matching
- Video call integration
- Subscription plans for premium features

## [1.0.0] - 2026-01-XX

### Added
- Initial release of OpSkl mobile app (iOS & Android)
- User registration and authentication
- Worker and client profiles
- Gig posting and browsing
- Real-time location-based matching
- In-app messaging
- Payment integration with Razorpay
- Wallet system
- Rating and review system
- Push notifications
- Biometric authentication for payments
- Admin dashboard for platform management
- Analytics and reporting
- Content moderation tools

### Security
- Row Level Security (RLS) on all database tables
- End-to-end encryption for sensitive data
- JWT-based authentication
- SSL/TLS for all communications
- PCI DSS compliant payment processing

## [0.9.0] - 2025-12-15 (Beta Release)

### Added
- Beta testing program launch
- Core gig economy features
- Basic payment integration
- User profiles
- Search and filters

### Changed
- Improved UI/UX based on alpha feedback
- Enhanced performance optimizations
- Better error handling

### Fixed
- Login issues on some Android devices
- Payment webhook timeout
- Location accuracy improvements

## [0.5.0] - 2025-11-01 (Alpha Release)

### Added
- MVP features for internal testing
- Basic authentication
- Simple gig posting
- Profile creation
- Mock payment flow

### Known Issues
- Limited to 50km radius for gigs
- No real-time notifications
- Basic UI design
